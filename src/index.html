Ray Tracer

A didactic ray tracer built with . to simulate objects under light transport. It has the following features:

  • Ray Generation: Generates primary rays from the camera's viewpoint.
  • Ray-Object Intersection: Computes intersections between rays and objects in the scene (spheres, planes, etc.).
  • Material Rendering: Implements basic material properties such as diffuse and reflective surfaces.

For a more detailed documentation please checkout the github link.

8-bit-cpu

This project is an implementation of an 8-bit CPU based on the Ahmes architecture, as outlined in the guide by Embedded Systems. It supports 24 instructions, uses an 8-bit data bus, and can address 256 bytes of memory. The CPU was developed using VHDL.
